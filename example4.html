TwoStateButton(id, upImage, downImage, ariaLabel, title, callback)
-- Works with touch, mouse and is accessible via keyboard.
-- If mouseup occurs outside of button, button goes up.
  • id is the id of the DOM element that is the button.
  • upImage is the path to the graphic displayed when user does not have mouse down.
  • downImage is the path to the graphic displayed when user presses mouse or touches button.
  • ariaLabel is the string used to describe the button for accessibility.
  • title is the string that displays near the button on mouseover.
  • callback is called after button is clicked.
The CSS style attached to the button must specify the width and height.
The CSS style attached to the button should also specify cursor:pointer
Example:
var button = new TwoStateButton("b1", "up.jpg", "down.jpg", "description 1", "description 2", function(e) {
    // Runs when the element with id="b1" is clicked.
    console.log(e.target.id);
}
